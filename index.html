Meu aprendizado

Durante esse primeiro módulo, denominado Fundamentos, tive contato com vários conteúdos. Nesse módulo foi possível ter contato e aprender tópicos básicos de comandos linux, tais como criar e remover diretórios. Aprendemos também como utilizar a ferramenta GitHub usando comandos tais como git status, git add, git commit e git push e git pull. Outro tema muito importante foi a introdução ao temo HTML e CSS. Aqui Aprendemos vários conceitos básicos e como utilizar as principais Tags do HTML. Fomos iniciados em pontos imporantes como o HTML semântico, bem como estilizar a nossa página HTML por meio do CSS Style.
